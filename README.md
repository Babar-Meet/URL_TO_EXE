# URL to EXE Builder

Convert any website URL into a Windows desktop EXE using Electron.

## Purpose
This project provides a local web interface where a user can:
1. Enter a website URL.
2. Set an EXE name.
3. Set a window title (header title shown when EXE runs).
4. Optionally upload a custom logo.
5. Generate and download a Windows EXE.

## Prerequisites
1. Windows OS.
2. Node.js LTS (includes npm).

## Get the Project
Choose one option.

### Option A: Download ZIP
1. Download the project ZIP.
2. Extract it to any folder.
3. Open the extracted folder.

### Option B: Git Clone
1. Open terminal in your preferred directory.
2. Run:

```bash
git clone https://github.com/Babar-Meet/URL_TO_EXE
```


## How to Run :
## Steps

1. Get the project:
- Download the ZIP and extract it, or
- Clone the repository:

```bash
git clone https://github.com/Babar-Meet/URL_TO_EXE
```

2. Run setup:

```bat
setup.bat
```

3. Run start:

```bat
start.bat
```

4. Open browser at:

`http://localhost:3000`

5. Enter details in the app:
- Website URL
- EXE Name
- Window Title
- Optional logo

6. Click `Generate EXE`.

7. When build is complete, click `Download EXE`.
3. If a build fails: check the error message shown in the app status box.
